🎯 3秒快速开始：

1. 获取OpenAI API Key: https://platform.openai.com/api-keys
2. 编辑 .env 文件，填入: OPENAI_API_KEY=sk-your-key-here
3. 运行启动脚本:
   • Windows: scripts\run_local.bat
   • Mac/Linux: bash scripts/run_local.sh
   • Docker: bash scripts/run_docker.sh
4. 访问: http://localhost:8000/docs

❓ 常见问题快速答案：

Q: 如何获取OpenAI API Key?
A: 访问 https://platform.openai.com/api-keys

Q: 支持哪些输入方式？
A: PDF上传、arXiv ID、DOI标识符

Q: 响应时间需要多久？
A: 首次8-15秒，缓存命中<2秒

Q: 是否支持离线使用？
A: 部分功能可以，但LLM分析需要OpenAI API

Q: 如何部署到生产环境？
A: 使用 bash scripts/deploy_tencent_cloud.sh 自动部署

更多问题请查看 README.md 或 DEPLOYMENT_CHECKLIST.md
